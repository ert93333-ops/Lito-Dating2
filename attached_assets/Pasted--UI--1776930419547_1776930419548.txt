핑크/로즈 톤, 라운드 카드, 인증 배지, 큰 CTA 버튼 덕분에

특히 한국·일본 교류 앱이라는 성격도 꽤 자연스럽게 녹아 있습니다.

좋은 점
1. 톤앤매너가 일관적이다
전체적으로 블러시 핑크 + 웜 화이트 + 소프트 그레이 조합이 잘 유지됨

버튼, 카드, 배지, 탭바가 같은 계열 안에서 움직여서 브랜드 인상이 안정적임

“로맨틱하지만 부담스럽지 않은” 분위기가 잘 남

2. 발견 / 프로필 화면의 감정 전달력이 좋다
프로필 사진을 크게 쓰는 방식이 좋음

발견 화면은 첫인상이 강하고, 프로필 상세로 넘어가도 분위기가 잘 이어짐

매칭 성공 화면도 감정 포인트가 분명해서 전환 순간이 살아 있음

3. CTA가 비교적 명확하다
대화 시작하기, 대화 계속하기, 채팅 같은 핵심 버튼이 눈에 잘 들어옴

주요 액션이 브랜드 컬러와 연결돼 있어서 사용자가 망설이지 않음

4. “신뢰”를 UI로 보여주려는 시도가 좋다
본인 인증, 얼굴 인증, 신분증 인증 같은 요소가 visible 하게 들어가 있음

이 앱의 핵심 가치 중 하나가 “안전한 연결”이라는 걸 UI가 말해줌

프로필 완성도/신뢰 강도 카드도 방향이 좋음

5. 기능이 많아도 구조는 어느 정도 잡혀 있다
채팅, 매칭, 프로필, 설정이 각자 역할이 분명함

프로필 홈에서 “지금 할 수 있는 것”처럼 행동 유도형 카드가 있는 건 좋음

매칭 화면도 추천 문장 + 프로필/채팅 CTA 구조가 이해 가능함

개선점
1. 정보 밀도는 조금 더 줄여도 좋다
특히 발견 화면 / 프로필 상세 / 매칭 카드에서 태그와 정보가 많이 보입니다.

2. 다국어 표현 방식은 더 정리할 수 있다
한일 서비스라 다국어가 강점이긴 한데, 지금은 화면에 따라

상대 소개는 번역본 우선

채팅도 원문과 번역이 동시에 계속 보이기보다 선택형이 더 깔끔함

예쁜 연핑크/연회색 계열을 쓰고 있어서,

프로필 홈 상단의 큰 그라데이션 영역은 무드는 좋지만,

5. 설정 화면은 기능 구조는 괜찮지만, 브랜딩 밀도는 약하다

화면별 추천 애니메이션

추천 모션
카드 진입: 아래에서 살짝 올라오며 fade in

좋아요/패스 스와이프: 손가락 이동에 따라 실시간 반응

하트 누름: 작은 scale up + soft pulse

포인트
좌우 스와이프에 맞춰 LIKE / PASS 피드백이 opacity로 반응하면 좋음

별/찜 기능은 골드 글로우를 아주 약하게만

버튼은 튕기지 말고 “눌리는 느낌” 위주로

탭 반응: 120~160ms

카드 진입: 220~280ms

카드 스와이프 settle: 240~320ms

여긴 감정 연출의 핵심이에요.

너무 축제처럼 하지 말고 “설레는 연결”처럼

추천 시간
카드 이동: 300~400ms

하트 pulse: 180ms × 2

CTA 등장: 200ms

3. 채팅 화면
채팅은 빠르고 자연스러워야 합니다.

추천 모션

받은 메시지: 부드러운 등장만

타이핑 표시: 점 3개 opacity loop

AI 추천 답장: 하단에서 시트처럼 올라오기

번역 보기 전환: crossfade

중요한 점

과한 애니메이션보다 **“즉시 반응 + 미세한 부드러움”**이 더 중요합니다.

메시지 등장: 160~220ms

번역 토글: 180~240ms

추천 답장 패널: 220~280ms
