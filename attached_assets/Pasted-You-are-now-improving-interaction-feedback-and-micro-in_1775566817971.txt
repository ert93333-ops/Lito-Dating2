==================================================
CURRENT PROBLEM
==================================================

The app feels too dull and lifeless.

When the user performs actions such as:
- tapping buttons
- like
- pass
- translation
- send
- open profile
- select tabs
- confirm actions

there is little or no visible feedback.
As a result, the app feels flat, boring, and low-energy.

Your task is to improve the interaction quality through micro-interactions, motion feedback, state transitions, and tactile response.

==================================================
1. GOAL
==================================================

Make the app feel:
- responsive
- alive
- tactile
- premium
- polished
- satisfying to use

But also:
- not noisy
- not childish
- not over-animated
- not game-like
- not flashy

The style should remain:
- premium
- warm
- modern
- elegant
- Korean/Japanese dating app appropriate
- Bumble-inspired in clarity, but softer and more refined

==================================================

Improve interaction feedback for these actions:

A. Button taps
- primary CTA buttons
- secondary buttons
- icon buttons
- chips/tags
- bottom nav items
- tab switches

- press down state
- release feedback
- subtle scale / opacity / elevation response
- immediate visual acknowledgment

B. Like action
- more satisfying active feedback
- icon response
- subtle burst / ripple / glow / highlight if appropriate
- elegant, restrained feedback
- premium feel, not cheesy

C. Pass action

G. Match / success moments

==================================================

- motion transition

- loading indicator
- success confirmation state
