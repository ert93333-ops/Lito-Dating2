# 최상위 원칙

- paywall은 기본 채팅을 막지 않는다
- paywall은 기본 번역을 막지 않는다
- 사용자의 명시적 AI 요청 없이 먼저 튀어나오지 않는다
- 신고/차단/삭제 근처에 두지 않는다
- "기본 채팅과 기본 번역은 계속 무료"를 반드시 명시

# 허용 위치

- 채팅방 AI 다듬기 버튼 탭 직후
- 첫 메시지 제안 요청 직후
- 프로필 자기소개 AI 개선 요청 직후
- 설정 > AI 코칭 > 크레딧 상점

# 금지 위치

- 온보딩 중
- 프로필 작성 직후 강제 팝업
- Discover
- Match 직후
- 번역 버튼/번역 결과 위
- 신고/차단/안전 경고 근처
- 삭제/계정 설정 플로우
