# 필수 체크리스트

## 1. 18+ age gate
- DOB 또는 self attestation 기반 성인 여부 판정
- 미성년자 차단
- 서버측 연령 상태 유지

## 2. 로그인 조합
- Apple / Google / Email
- iOS에서 Google 제공 시 Apple 동시 제공

## 3. Terms / Community / Child Safety / Support
- 가입 전 Terms / Privacy / Community 동의
- Help / Safety / Contact 접근 가능
- child safety / CSAE 대응 문서 공개 가능 구조

## 4. 신고 / 차단 / 운영자 제재
- profile/photo/message report
- block user
- moderation log
- suspend/ban/remove action

## 5. 매칭 구조
- 상호 매칭 후에만 1:1 채팅
- 공개 피드/커뮤니티 MVP 제외

## 6. AI 기능
- 번역 first-use opt-in
- 코칭 first-use opt-in
- AI 결과 신고 가능

## 7. 계정 삭제
- 앱 내 Delete Account
- 웹 삭제 요청
- 즉시 비노출/세션 종료

## 8. 결제
- 모든 디지털 상품은 IAP / Play Billing
- 복원/검증 경로 존재

## 9. 개인정보/메타데이터
- Privacy Policy URL
- App Privacy / Data Safety 정합성
- SDK/벤더 인벤토리

## 10. 리뷰어 접근
- 리뷰 계정 2개 이상
- 매칭/채팅/번역/신고/차단/삭제까지 검수 가능
