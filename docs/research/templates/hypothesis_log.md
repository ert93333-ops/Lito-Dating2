# Hypothesis Log

LITO 제품 가설의 누적 기록.
새 가설은 최하단에 추가한다.
검증 완료 시 상태를 업데이트하고 결과와 다음 액션을 기입한다.

담당: 01_리서치팀 (수립) / 02_제품팀 (우선순위) / 00_지휘본부 (검증 완료 확인)

---

## 형식

| 필드 | 설명 |
|---|---|
| ID | H-NNN (순번) |
| 분류 | 성장 / 유지 / 핵심기능 / 수익 / 신뢰 |
| 가설문 | "우리는 [대상]이 [행동]을 할 것이라고 믿는다. 왜냐하면 [근거]." |
| 검증 방법 | 인터뷰 / A/B 테스트 / 데이터 분석 / 리뷰 마이닝 |
| 성공 지표 | 어떤 수치가 어느 수준이면 가설이 맞다고 볼 것인가 |
| 상태 | 미검증 / 검증중 / 확인됨 / 기각됨 |
| 검증 결과 | 검증 완료 후 작성 |
| 다음 액션 | 검증 결과에 따른 후속 조치 |
| 관련 문서 | 근거 파일 경로 |
