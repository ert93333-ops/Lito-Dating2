# Flow

Welcome
-> Age Gate
-> Auth Method Select
-> Required Consents
-> Email Verification / Auth Complete
-> Profile Basics
-> Profile Intent & Language
-> Profile Translation Preview
-> Contact Block Opt-in
-> Safety Intro
-> Discover

# 고정 규칙

- AI 데이터 처리 동의는 여기서 묶지 않는다
- 번역/코칭 동의는 실제 기능 사용 시 받는다
- 프로필 번역 미리보기는 publish 전 필수
- 연락처 차단은 opt-in
- 신고/차단/삭제 존재를 온보딩에서 알려준다
