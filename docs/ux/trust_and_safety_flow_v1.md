# Safety Surfaces

- Discover card / profile detail: report, block
- Match / Chat: report, block, unmatch
- Trust Center
- Contact Block Manage
- Settings / Account & Privacy
- Delete Intro / Re-auth / Web Handoff

# 원칙

- 신고/차단은 숨기지 않는다
- block는 즉시 작동한다
- delete는 앱 내 시작 + 웹 요청으로 이어진다
- unsafe 상황에서 AI보다 safety CTA 우선
- 삭제 플로우 중 결제/리텐션 팝업 금지
